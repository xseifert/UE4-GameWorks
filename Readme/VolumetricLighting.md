How to Use
----------

* NVIDIA Volumetric Lighting only supports DirectX 11 by now.
* Please see the **MediumDesign.pdf** and **UE4_VL_Overview.pdf**.
* There is a **NvVlSamples** project. You could find out how the authoring works.
